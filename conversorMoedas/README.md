Conversor de Moedas:
O Conversor de Moedas é uma aplicaçăo desenvolvida em Java com Spring que permite aos usuários converter valores 
entre diferentes moedas de forma rápida e eficiente.

Funcionalidades:
Conversăo entre cinco moedas principais:
Real (BRL)
Euro (EUR)
Dólar (USD)
Menu interativo para selecionar a moeda de origem, destino e valor a ser convertido.
Integraçăo com a API ExchangeRate-API para obter taxas de câmbio em tempo real.

Tecnologias Utilizadas:
Java 17
Spring Framework
ExchangeRate-API
GSON (manipulaçăo de JSON)
Dotenv (gerenciamento de variáveis de ambiente)
